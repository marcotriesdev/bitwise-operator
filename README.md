# Briefing
A fantasy pixel art game made with python and Raylib to practice procedural programming. 
The player input and core inventory are managed through bitwise operations to practice efficiency and binary shifting/masking. Asyncio code is included to allow easy porting to web besides plans on exporting executable version for pc.
